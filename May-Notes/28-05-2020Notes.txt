	           Bootstrap Accordion
	           ==================
- It is a collection of cards which you can switch between display dynamically. It uses collapse and card with accordion.
- It requires ".accordion" class

Note: All cards must be in accordion with collapse.
         All collapse container must map to 
	"data-parent=accordionId"

	                Bootstrap DropDowns
- All use bootstrap dropdown for navigation
- Don't use bootstrap dropdown for interactions like adding, removing items dynamically. For that you better prefer using HTML <select>


Classes:
.dropdown	It creates a dropdown
.dropdown-menu	It is a collection of items in dropdown
.dropdown-item	It refers individual dropdown item.
.dropdown-toggle	It uses ON/OFF effects
.dropdwon-divider	It sets a horizontal line as divider

Note: Bootstrap Dropdown require "popper.js" which you can get from "bootstrap.bundle.js"

Plugins:
data-toggle
data-target
